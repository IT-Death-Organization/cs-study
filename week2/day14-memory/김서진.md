## 1. 메모리 단편화는 왜 발생하는가?
메모리 단편화(Fragmentation)는 사용 가능한 메모리 총량은 충분하지만, 연속된 공간이 부족하여 할당이 실패하는 현상이다.
메모리 할당/해제가 반복되면서 다음과 같은 상황이 생긴다:
[사용][사용][비어있음][사용][비어있음][사용]
- 프로세스가 종료되면서 중간중간 구멍(hole)이 생김
- 새 프로세스는 연속된 큰 공간을 요구
- 하지만 공간이 흩어져 있어 할당 실패
즉, 동적 메모리 관리 + 연속 할당 방식이 결합될 때 필연적으로 발생
